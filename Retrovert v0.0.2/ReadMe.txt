Step-By-Step Instructions: How To Apply Or Install Presets

Quick Note 1: You should apply certain effects and adjust a few colors to your image/video before rendering in this present for extra VHS features that are outside these two software's! 

1. NTSC-QT:

1. Download the Retrovert file. (Skip this since you already have it, you're on it right now.)
2. Go to Presets>NTSC-QT>(and select one of the presets you would like.)
3. Open the preset file and copy everything by using this short-cut: CRTL+A CRTL+C.
4. Open NTSC-QT by running it 'as administrator'. (And make sure you have your image/video opened in it.)
5. Locate the 'Import / Export preset' button. And open it. (Which should be located at the bottom corner-left.)
6. Once in, select everything and delete, then paste in the preset. You can use this shortcut: CRTL+A Back CRTL+V.
7. Once pasted, you can now press or click 'Save'. And can safely exit the Dialog prompt. You should be able to see the effects applied.
8. Enjoy! Thank you for using our presets!

2. NTSC-RS:

1. Download the Retrovert file. (Skip this since you already have it, you're on it right now.)
3. Open/Run NTSC-RS. (And make sure you have your image/video opened in it.)
4. Locate the 'Load from' button. And open it. (Which should be located at the bottom corner-left.)
5. Once the file explorer prompt shows up, find the preset file you would to use. Once you have found it, click it to select it and press 'Open' on the bottom.
6. Enjoy! Thank you for using our presets!
